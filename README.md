# shinshin-impact 6.4 (Genshin Impact Cheeto)

**cheeto for gensen(Shinshin) impact**

this is NOT a calc app
(Calc is short for calculator btw). 

## CURRENT features
- God Mode - Take no damage
- Multi Hit - Hit enemies multiple times per attack
- Infinite Stamina - Never run out of stamina
- Infinite E skill - Spam your E
- Infinite Ultimate - Spam your burst
- Sprint CD
- Attack Multiplier
- noclip(WIP)
- Time Scaling for the entire game
- Skip Video cutscenes
- Teleport(with custom group teleports)
